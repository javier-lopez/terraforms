# HAPROXY + NGINX BACKENDS

## Overview

Simple infrastructure that consists of two Nginx servers that are load balanced
by an HAProxy server.

## Usage

```
$ terraform init
$ terraform plan
$ terraform apply
$ terraform destroy
```
